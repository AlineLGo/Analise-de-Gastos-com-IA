# 📊 Análise de gastoss automatico com IA.

Projeto de análise de gastos com Python + Inteligência Artificial.

## 🚀 Funcionalidades
- Classificação automática de gastos
- Geração de relatório financeiro
- Insights com IA (Gemini)

## 🛠️ Tecnologias
- Python
- Pandas
- Google Colab
- API Gemini

## 📈 Resultado
O sistema analisa seus gastos e gera:
- Resumo financeiro
- Categoria com maior gasto
- Dicas de economia com IA
